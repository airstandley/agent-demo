Basic demo of an agent harness.

Built by going through the lessons in pguso's 'Agents From Scratch' repo: https://github.com/pguso/agents-from-scratch/tree/main/lessons

Setup:
1. Install Ollama(https://ollama.com/download) locally
2. Pull the relevant models.
```ollama pull phi3:mini```
3. Install Poetry
```pip install poetry```
4. Install project dependencies
```poetry install```

Running the Demo:
```poetry run python main.py```
